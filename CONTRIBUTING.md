# 🤝 Mitwirken an TechRZN / Contributing to TechRZN

[Deutsch] | [English below]

Vielen Dank, dass du helfen möchtest, die **TechRZN-Blocklist** noch besser zu machen! 

### 🛡️ Wie kann ich helfen?
1. **Neue Domains melden:** Wenn du eine Seite findest, die Werbung schaltet oder gefährlich ist, erstelle bitte ein [Issue](https://github.com/SmokingBull/TechRZN-Blocklist-Collection/issues).
2. **Fehlblockierungen (False Positives):** Wenn eine wichtige Seite nicht mehr funktioniert, sag uns Bescheid, damit wir sie auf die Whitelist setzen können.
3. **Verbesserungen:** Hast du Ideen für die GitHub Actions oder die Struktur? Pull Requests sind willkommen!

### 📝 Regeln
* Bitte bleib höflich und sachlich.
* Gib bei Meldungen immer an, welche Domain betroffen ist und was genau das Problem ist (z.B. "Werbung wird trotz Filter angezeigt").

---

## 🇺🇸 English Version

Thank you for your interest in improving the **TechRZN Blocklist**!

### 🛡️ How can I contribute?
1. **Report new domains:** If you find a site that serves ads or is malicious, please open an [Issue](https://github.com/SmokingBull/TechRZN-Blocklist-Collection/issues).
2. **False Positives:** If a legitimate website stops working, let us know so we can add it to the whitelist.
3. **Improvements:** Do you have ideas for GitHub Actions or the repository structure? Pull requests are welcome!

### 📝 Guidelines
* Please be respectful and constructive.
* When reporting, always specify the affected domain and describe the issue (e.g., "Ads are still visible despite the filter").
